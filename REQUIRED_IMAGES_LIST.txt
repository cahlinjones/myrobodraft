==============================================
IMAGE SPECIFICATIONS:
==============================================

Format: WebP (.webp)
Recommended Size: 800x800px to 1200x1200px
Aspect Ratio: 1:1 (square) or 4:3
Quality: High (suitable for web)
Background: Clean/white background preferred for product shots

==============================================
WHERE TO PLACE IMAGES:
==============================================

All images should be placed in the /images/ directory of your website.

Full path example: /images/mammotion-luba-mini-awd-main.webp


==============================================
NOTES:
==============================================

✓ Each product page uses 4 thumbnails in the gallery
✓ The main image is also used as the first thumbnail
✓ If you only have 4 images per product total, use the main image twice (once as main, once as thumbnail)
✓ Action shots showing the mower on a real lawn are most effective for conversions
✓ Detail shots should highlight unique features (motors, navigation sensors, control panel, etc.)
